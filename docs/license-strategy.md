# License strategy

Codex Coding OS uses the Mozilla Public License 2.0.

## Current posture

MPL-2.0 is a file-level copyleft license. It allows commercial use, private use,
modification, distribution, and use inside larger works, while keeping covered files
and modifications to covered files under MPL-2.0 when they are distributed.

This is an open-source posture that allows commercial use under the license terms.

## Why MPL-2.0

MPL-2.0 fits this repository better than a broad copyleft license because the repo
is mostly skills, templates, scripts, documentation, and instruction files. The goal
is to keep improvements to those covered files shareable without forcing unrelated
applications created with the pack to inherit the same license.

## Commercial use

Commercial use is allowed under `LICENSE.md`. `COMMERCIAL-USE.md` explains the
practical commercial-use boundary and does not create a separate paid-license
requirement.

## Output boundary

Applications, codebases, documents, prompts, and business outputs created by using
this pack are not automatically part of this repository. Copied or modified covered
files from this repository remain subject to MPL-2.0.

## Third-party materials

Third-party files and referenced external repositories may have their own licenses.
Do not assume the top-level license relicenses third-party material that is separately
licensed or only referenced.

Before public releases, review:

- `NOTICE.md`
- `THIRD_PARTY_SKILLS.md`
- bundled `LICENSE.txt` and `NOTICE.txt` files
- `external-skills/manifest.json`
- copied templates or references that came from outside this repository

If provenance is unclear, keep the material linked as reference-only until its license
and redistribution terms are reviewed.
